# eeclass pdf downloader

An extension to download pdf from eeclass.

## Usage
### Edge
1. Enter URL `edge://extensions` in your edge browser

2. click `載入解壓縮` and import the whole directory
<img width="1327" height="241" alt="image" src="https://github.com/user-attachments/assets/bacab544-c797-4214-a262-28e08dfec8c2" />

3. Scroll down and make sure it's open
<img width="850" height="132" alt="image" src="https://github.com/user-attachments/assets/0a90f587-7e9f-42dd-be11-e99ae4b2e227" />

4. Go to eeclass, open the slid that you want to download, Open "eeclass PDF Downloader"
<img width="1269" height="733" alt="image" src="https://github.com/user-attachments/assets/c161f5a5-97d7-4562-8ad0-8686ee1ea2f3" />

5. Click "開始轉換 PDF"
<img width="1208" height="728" alt="image" src="https://github.com/user-attachments/assets/bd4cdd60-6ea6-49b4-a3dd-6feaa0686575" />

6. Finish!
